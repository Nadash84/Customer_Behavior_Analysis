# Customer Behavior Analysis

## Overview
This project is an end-to-end Data Analytics project based on a Retail Customer Behavior and Shopping Trends dataset. The goal was to transform raw customer data into meaningful business insights using Python, SQL Server, Power BI, and Gamma.

The project follows a real-world analytics workflow, including data cleaning, exploratory data analysis (EDA), SQL querying, dashboard creation, and presentation of findings.

---

## Project Objectives
- Clean and prepare raw customer data.
- Explore customer purchasing patterns and shopping trends.
- Perform business analysis using SQL queries.
- Build an interactive Power BI dashboard.
- Present key insights and recommendations.
- Demonstrate an end-to-end analytics workflow suitable for a professional portfolio.

---

## Tools & Technologies
- **Python (Google Colab)** – Data cleaning and preparation
- **Pandas** – Data manipulation and analysis
- **SQL Server (SSMS)** – Data storage and SQL analysis
- **Power BI** – Dashboard creation and visualisation
- **Gamma** – Presentation and reporting
- **GitHub** – Version control and project documentation

---

## Project Workflow

### 1. Data Cleaning & Preparation
- Imported dataset into Google Colab
- Standardised column names
- Checked data types
- Created new features such as age groups
- Removed inconsistencies and prepared data for analysis

### 2. Exploratory Data Analysis (EDA)
- Examined customer demographics
- Analysed purchasing behaviour
- Identified trends and patterns
- Investigated customer spending habits

### 3. SQL Analysis
Key business questions included:
- Revenue generated by male vs female customers
- Impact of discounts on spending
- Highest-rated products
- Customer subscription behaviour
- Revenue contribution by age group
- Product and category performance

### 4. Dashboard Development
Created an interactive Power BI dashboard to visualise:
- Revenue trends
- Customer demographics
- Product performance
- Shopping behaviour insights
- Customer segmentation

### 5. Presentation & Reporting
A professional presentation was created in Gamma to communicate:
- Key findings
- Business insights
- Recommendations
- Data-driven decision making opportunities

---

## Key Results
- Identified top-performing products and categories.
- Analysed customer spending behaviour.
- Evaluated the effectiveness of discounts and subscriptions.
- Segmented customers based on purchasing behaviour.
- Generated actionable insights to support business decisions.
